# Bommy's Cake — Luxury Cake Shop Website

A fully designed luxury cake shop landing page for Bommy's Cake with dark gold styling, sectioned content, menu ordering, cart checkout, WhatsApp & email order notifications, and payment selections.

## Files
- `index.html` — full website code with styling and ordering logic.
- `.github/copilot-instructions.md` — workspace instructions for Copilot.

## Local preview
1. Install Node.js if you don't already have it.
2. Open a terminal in this folder.
3. Run `npm install`.
4. Run `npm start`.
5. Open `http://127.0.0.1:4173` in your browser if it does not open automatically.

## Publish live with GitHub Pages
1. Create a GitHub repository for this project.
2. In your terminal, run:
   - `git init`
   - `git add .`
   - `git commit -m "Initial Bommy's Cake site"`
   - `git branch -M main`
   - `git remote add origin https://github.com/<your-username>/<repo-name>.git`
   - `git push -u origin main`
3. In GitHub, go to `Settings` → `Pages`.
4. Set the source to the `main` branch and `/ (root)` folder.
5. Save and wait for GitHub to publish the site.
6. The live URL will look like: `https://<your-username>.github.io/<repo-name>/`.
